# DiaSmart — Diabétologie Intelligente 🇨🇲

**DiaSmart** est une application mobile gratuite de gestion du diabète conçue pour le Cameroun et l'Afrique Sub-Saharienne.

## Fonctionnalités principales
- 📊 Suivi glycémie + HbA1c avec graphiques
- 🤖 ROLLY — Assistant IA spécialisé en diabétologie
- 👨‍⚕️ Partage médecin-patient sécurisé
- 🍽️ Analyse de repas avec photo (Cameroon-aware : ndolè, plantain, eru...)
- 📱 100% offline-first, sync automatique
- 🌍 7 langues : FR, EN, AR, Pidgin, Duala, Bassa, Fulfulde, Dii
