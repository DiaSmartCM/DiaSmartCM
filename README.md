## ✨ Fonctionnalités principales

- 📊 **Suivi glycémie + HbA1c** avec graphiques et statistiques
- 🤖 **ROLLY** — Assistant IA spécialisé en diabétologie (ton camerounais, aliments locaux)
- 👨‍⚕️ **Partage médecin-patient** sécurisé (consentement RGPD explicite)
- 🍽️ **Analyse de repas par photo** — connaît les plats locaux : ndolè, plantain, eru, manioc, igname…
- 📱 **100 % offline-first** — fonctionne sans réseau, sync automatique au retour
- 🌍 **7 langues** : Français, English, العربية, Pidgin Kamtok, Duala, Bassa, Fulfulde, Dii
- 🔒 **Conforme RGPD + loi camerounaise 2024** sur les données personnelles

---

## 🩺 Pour qui ?

- **Patients diabétiques** : suivi quotidien, rappels médicaments, carnet de bord
- **Médecins** : tableau de bord patients, validations IA, génération d'ordonnances/comptes-rendus
- **Aidants** : mode famille pour suivre un proche en lecture seule

## 🛠️ Stack technique

`Kotlin` · `Jetpack Compose` · `Hilt` · `Room (SQLCipher)` · `Firebase Spark` · `Vercel` · `Gemini 2.5 Flash`

100 % gratuit à héberger — architecture pensée pour le Free tier (Firebase Spark + Vercel + Supabase).
